Employees

Up 
get /api/employees
Получение списка пользователей (apiEmployeesGet)
Получение списка пользователей с параметрами для поиска (поиск по полям, сортировка и т.п.)

Query parameters

name (optional)
Query Parameter — Имя сотрудника
surname (optional)
Query Parameter — Фамилия сотрудника
patronymic (optional)
Query Parameter — Отчество сотрудника
position (optional)
Query Parameter — Должность сотрудника
order (optional)
Query Parameter — Поле сортировки (id, name, patronymic, surname, position, birthday, phone)
sort (optional)
Query Parameter — Вид сортировки (asc, desc)

Return type

List
